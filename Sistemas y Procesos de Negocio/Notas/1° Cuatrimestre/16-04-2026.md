# Sistema de informacion

son procesos para recolectar datos, noticioas, conocimiento. 
Conjunto de procesos para recolectar y distribuir la informacion en tiempo y forma para la toma de decision

## Sistema
Un conjunto de elementos interrelacionados con un objetivo en comun

Los elementos pueden ser personas u objetos, es universal no importasn de donde vengas. 
Ej: sistema nervioso, sistema operativo, sistema numeroico

Todo lo que esta relacionado esta dentro del limite/frontera del sistema.

Sistema microentorno y macroentorno

Tiene que tener un objetivo comun

Todo sistema tiene entradas, se procesa (caja negra si no se sabe como se procesa)(Caja blanca si sabes como es el proceso), retroalimentacion (correccion/control de calidad) y por ultio salida

Sistema abierto, cerrado, aislado
### Sistema abierto: interactuan con el entorno, entra y sale energia y materia

### Sistema cerrado: solo intercambia la energia, no la materia

### Sistema aislado: no intercambia ni materia ni energia
